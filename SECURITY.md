# 🔐 توثيق الحماية لموقع الآلة الحاسبة

هذا الملف يوضح كيفية تأمين موقع الآلة الحاسبة وحماية تعليقات المستخدمين ضد الهجمات والتهديدات المختلفة مثل XSS والسبام. تم تصميم الآليات التالية لضمان الأمان:

---

## 1. تنظيف النصوص (Sanitize Input)

يتم تنظيف النصوص المدخلة من قبل المستخدمين لمنع هجمات **XSS** (حقن أكواد JavaScript). يتم استبدال الأحرف الخاصة (مثل `<`, `>`, `&`, `"`, `'`) برموز HTML المناسبة.

**الكود**:
```javascript
export function sanitizeInput(input) {
  if (typeof input !== "string") return "";
  return input
    .replace(/&/g, "&amp;")
    .replace(/</g, "&lt;")
    .replace(/>/g, "&gt;")
    .replace(/"/g, "&quot;")
    .replace(/'/g, "&#039;");
}
